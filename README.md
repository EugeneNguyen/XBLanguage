## Installation

XBLanguage is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

    pod "XBLanguage"

## How to use

I. Import XBLanguage.h everywhere you want.
II. Using following code

